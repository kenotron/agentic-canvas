# Infinite Canvas React App

A React + TypeScript infinite canvas application built with Vite, using Legend State v3 beta for state management.

## State Management

This project uses **Legend State v3 Beta** as its state management library.

Install the beta release:

```bash
pnpm add legend-state@beta
```

Adjust the store setup in `src/state/canvasStore.ts` as needed to match the v3 API.

## Getting Started

Install dependencies:

```bash
pnpm install
```

Start development server:

```bash
pnpm dev
```

Build for production:

```bash
pnpm build
```

Preview production build:

```bash
pnpm preview
```
